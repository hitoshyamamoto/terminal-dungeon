## üìã Document Descriptions

### START_HERE.md
**The main entry point for new users.** Covers installation, basic gameplay, and links to all other documentation. Start here if you're new!

### QUICKSTART.md
**5-minute guide** to get from installation to playing your first game. Perfect for experienced users who want to jump right in.

### INSTALL.md
**Detailed installation guide** covering Node.js setup, dependencies, building, and troubleshooting common issues across different platforms.

### LAN_SETUP_GUIDE.md
**Network configuration guide** for setting up multiplayer games. Covers firewall rules, port forwarding, and troubleshooting connection issues.

### TESTING_GUIDE.md
**Complete testing documentation** including automated test suites, game simulator, and how to create custom tests for development.

### PROJECT_SUMMARY.md
**Technical overview** of the entire project, including architecture, implementation details, feature checklist, and development guidelines.

### commands.md
**Complete reference** of all 38 commands with syntax, descriptions, examples, and when to use each command.

### rules.md
**Full gameplay rules** including turn phases, combat mechanics, death/respawn, victory conditions, and advanced strategies.

### card_interactions.md
**Card mechanics guide** explaining how different card types work, interact with each other, and affect gameplay.

### networking.md
**Network setup & troubleshooting** covering UDP discovery, TCP connections, firewall configuration, and common network issues.

### protocol.md
**Technical specification** of the network protocol, message formats, state synchronization, and reconnection logic.
